# Charles Tsao - Personal Portfolio

A professional, responsive portfolio website built with HTML5, CSS3, and Vanilla JavaScript. This project showcases my skills, experience, and projects as a Senior Data Science Analyst.

## Features

-   **Responsive Design**: Fully responsive layout that works seamlessly on mobile, tablet, and desktop devices.
-   **Clean Aesthetics**: Modern, illustration-based design with a focus on typography and whitespace.
-   **Interactive Elements**: Smooth scrolling, hover effects, and a mobile navigation menu.
-   **Semantic HTML**: Built with accessibility and SEO in mind.
-   **Custom Sections**:
    -   **Hero**: Introduction and social links.
    -   **Skills**: Grid layout of technical skills.
    -   **Experience**: Timeline of professional experience.
    -   **About Me**: Personal bio and photo.
    -   **Projects**: Showcase of featured projects with a dark theme and alternating layout.
    -   **Contact**: Custom contact form and information.

## Tech Stack

-   **HTML5**
-   **CSS3** (Custom Properties / Variables)
-   **JavaScript** (Vanilla)
-   **Font Awesome** (Icons)
-   **Google Fonts** (Sora)
